# 📊 Global Superstore Sales Dashboard (Power BI)

## 📌 Project Overview
This project analyzes a global retail dataset over 4 years using Power BI.  
The goal was to perform exploratory data analysis (EDA) and forecast future sales.

---

## 🔍 Key Features
- Interactive dashboard with filters (Region & Category)
- Key metrics:
  - Total Sales
  - Total Orders
  - Average Sales per Order
- Sales analysis by region and category
- Time series analysis
- 7-day sales forecast

---

## 📈 Insights
- Technology is the top-performing category
- Sales show periodic spikes over time
- Forecast indicates stable short-term sales trends

---

## 🛠 Tools Used
- Power BI
- DAX (Data Analysis Expressions)
